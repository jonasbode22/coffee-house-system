# ☕ Coffee House System

Sistema de Gestão para Cafeteria desenvolvido como projeto didático da disciplina Projeto de Software.

## Objetivo

O sistema tem como objetivo apoiar o gerenciamento de uma cafeteria, permitindo controlar clientes, produtos, pedidos, estoque e informações gerenciais.

## Funcionalidades

- Login de usuários
- Cadastro de clientes
- Cadastro de produtos
- Registro de pedidos
- Controle de estoque
- Relatórios gerenciais
