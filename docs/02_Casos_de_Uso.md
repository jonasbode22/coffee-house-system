# Casos de Uso

Este documento apresenta os principais casos de uso do Sistema de Gestão para Cafeteria.

## Atores

- Teste
- Administrador
- Atendente
- Gerente

## Casos de Uso

### UC01 – Realizar Login - OK - REALIZADO COM SUCESSO

Permite que o usuário acesse o sistema utilizando suas credenciais.

---

### UC02 – Cadastrar Cliente

Permite registrar novos clientes para realização de pedidos.

---

### UC03 – Cadastrar Produto

Permite incluir novos produtos no catálogo da cafeteria.

---

### UC04 – Registrar Pedido

Permite registrar pedidos realizados pelos clientes.

---

### UC05 – Controlar Estoque

Permite acompanhar entradas, saídas e disponibilidade dos produtos.

---

### UC06 – Emitir Relatórios

Permite gerar relatórios gerenciais para acompanhamento das vendas.
