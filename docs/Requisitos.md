# Documento de Requisitos

Este documento apresenta os principais requisitos do Sistema de Gestão para Cafeteria.

## Requisitos Funcionais

- RF01 – Permitir login de usuários.
- RF02 – Permitir cadastro de clientes.
- RF03 – Permitir cadastro de produtos.
- RF04 – Permitir registro de pedidos.
- RF05 – Permitir controle de estoque.
- RF06 – Permitir emissão de relatórios gerenciais.

## Requisitos Não Funcionais

- RNF01 – O sistema deve possuir interface simples e intuitiva.
- RNF02 – O sistema deve proteger o acesso por meio de autenticação.
- RNF03 – O sistema deve armazenar os dados de forma organizada.
- RNF04 – O sistema deve permitir manutenção e evolução futura.
