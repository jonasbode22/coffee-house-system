# Plano de Testes

## Objetivo

Definir os critérios para validação das funcionalidades do Sistema de Gestão para Cafeteria.

## Funcionalidades a serem testadas

- Login
- Cadastro de Clientes
- Cadastro de Produtos
- Registro de Pedidos
- Controle de Estoque
- Relatórios

## Tipos de Testes

- Testes Funcionais
- Testes de Interface
- Testes de Validação
- Testes de Regressão

## Critério de Aprovação

A funcionalidade será considerada aprovada quando atender aos requisitos especificados e não apresentar falhas durante a execução dos testes.
